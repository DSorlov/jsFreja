Name Type Description

The minimum requested level.
ConfirmationMethod ConfirmationMethod

The confirmation method.
UserAttributes Array.<UserAttributes>

The user attributes to return.
RelyingPartyId string

The relying party ID (used by integrators only).
Example
const freja = new FrejaAPI(APIMode.TEST, 'path/to/pfx', 'password', 'path/to/cacert');
var result = await freja.initAuthRequest('SE199912121212');
if (result.isOk) {
 console.log(result.token);
 console.log(result.qrCodeUrl());
} else {
 console.error(result.message);
}
